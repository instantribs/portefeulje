Hierdie is die portefeulje van Ruben van Quickelberger 

Hierdie README bevat nie alleenlik al die informasie oor die portefeulje nie! Lees gerus deur die dokumentasie leers in die Dokumentasie folder om 'n oorsig van die portefeulje te kry

Daar is ook opgedateerde skermskote in die Screenshots folder van al die webblaaie

Tydens inspeksie van die webtuiste sal u die staf sign-in bladsy teekom en nie kan verder navigeer sonder om in te teken nie
Waneer u by die punt kom, raadpleeg asb die HTML of JS bronkode om hulpvolle kommentaar te vind met werkende login details

Dit is die moeite werd om die Stock blad twee keer te aanskou, een keer as 'n admin, die ander keer nie. Die bladsy sal verskillend lyk

Die Staff Info blad sal nie gebruikers inlaat wat nie 'n admin op die sisteem is nie

Al die Staf-sy blaaie sal nie enige gebruiker inlaat wat nie ingeteken is nie, so begin asb by die Sign-in blad.

Speel gerus die speletjie op die Fun blad, ek sal dit sterk aanbeveel. U gaan moet vinnig wees!
